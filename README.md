# uni-app 项目初始化

### Project setup
```
yarn install || npm install

```

### Project scripts
```
npm run new:comp
npm run new:api
npm run new:view
npm run new:store

```

### Compiles and hot-reloads for development
```
npm run serve
npm run dev:[mp-weixin]

```

### Compiles and minifies for production
```
npm run build
npm run build:[mp-weixin]
